## 📚 简介

### 这是什么？

这是一个为 [龙之生存 Mod](https://www.curseforge.com/minecraft/mc-mods/dragons-survival) 附属数据包开发提供 VSCode 智能补全支持的 MCDoc 工具包，在 [Spyglass 插件](https://github.com/SpyglassMC/Spyglass) 下工作，适用于 Minecraft 1.21.1 版本。早期版本中的大部分 MCDoc 文件源自 [dragonsurvivalteam.github.io](https://dragonsurvivalteam.github.io/partners/)。

### 为什么创建这个项目？

为方便国内制作者，创建了这个本地化补全工具包，解决网络访问和语言障碍问题，希望能帮助到有需要的制作者。

### 包含哪些内容？

本工具包在[龙之生存官方数据包生成器](https://dragonsurvivalteam.github.io/partners/)的基础上，对缺失的模块和功能进行了补充与修正，提供了完整的中文注释和适当的 Wiki 链接，并调整了部分结构以适配 [Spyglass 插件](https://github.com/SpyglassMC/Spyglass)。

## ✨ 特性

- 🇨🇳 **面向国内开发者** - 完整的中文注释和适当的 [我的世界 Wiki](zh.minecraft.wiki) 链接
- 🎯 **数据包 JSON 补全** - 能力、缺陷、物种、表情等
- 🎨 **资源包 JSON 补全** - 龙魂图标、生物模型、皮肤组件等
- 🐉 **龙生 Mod 专属支持** - 命令补全、自定义谓词、成就触发器等
